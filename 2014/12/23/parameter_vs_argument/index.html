Parameter / Argument 차이

23 December 2014

Parameter는 함수의 정의에 사용되는 것, Argument는 함수의 호출에 사용되는 것.

#include<stdio.h>

void func(int parameter) { // parameter = 매개변수. 호출되면서 전달받은 것.
    printf(" %d ", parameter);
}

int main() {
    func( 3 );   // argument = 인자. 함수를 호출할 때 전달하는 것.
}
