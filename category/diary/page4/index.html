Emoji Plugin 달았다. 신난다

예전에는 이모지 싫어했었는데, Github 쓰면서 자꾸 보다 보니 정이 들었다.

Jekyll 블로그에 Emoji Plugin 설치하기

https://github.com/jekyll/jemoji 참고

  1. GemFile에 추가
gem 'jemoji'
  1. _config.yml에 추가
gems:
- jemoji
  1. 예쁘게 보이도록 적당히 스타일을 추가해줘도 된다.
.emoji {
margin: 0px 2px 3px 2px;
}
